Análise e Modelagem de Sistemas

Você será capaz de especificar integralmente o comportamento do sistema de software a ser desenvolvido como um documento denominado SRS (Software Requirements Specifications). Avalie ainda os documentos SRS para garantir a qualidade do produto e desenvolva o projeto de conclusão de curso.

Desenvolvimento de Aplicações Distribuídas

Construa aplicações gráficas na linguagem C# (CSharp), aplicando os conceitos de orientação a objetos. Você irá utilizar as funcionalidades providas pelo arcabouço ADO.NET para acessar bases de dados, agilizando o desenvolvimento de aplicações através do uso de um ambiente integrado de desenvolvimento (IDE).

Seja capaz de codificar utilizando boas práticas de estilo de programação, escolhendo o servidor de aplicação que melhor atenda as necessidades de uma aplicação Web.

Estrutura de Dados

Desenvolva estruturas de dados lineares (listas, pilhas, filas) e não lineares (árvores), utilize estruturas de dados para solucionar problemas computacionais, escolhendo as mais adequadas para cada situação e construindo estruturas de dados orientadas a objetos.
